# NxGV Secondary School STEM Module

Welcome to the Next Generation Vehicle (NxGV) STEM Program.

This repository contains learning materials, practical activities, wiring diagrams, and Arduino/ESP32 code for secondary school students to learn autonomous vehicle technology.

---

- Vehicle architecture
- ESP32 controller
- DC motor
- Servo steering
- IR sensors
- Ultrasonic sensor
- Hardware connections
- Hardware testing

## Required Hardware

- ESP32 Development Board
- L298N Motor Driver
- DC Motor
- Servo Motor
- 5 IR Sensors
- HC-SR04 Ultrasonic Sensor
- Battery Pack
- Autonomous Vehicle Chassis

---

## Getting Started

1. Assemble the vehicle.
2. Complete Module 1.
3. Verify all hardware functions correctly.
4. Continue to Module 2.
