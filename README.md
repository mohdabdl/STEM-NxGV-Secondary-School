# NxGV Secondary School STEM Module

Welcome to the Next Generation Vehicle (NxGV) STEM Program.

## Learning Path

1. Hardware Introduction
2. Motor Control
3. Steering Control
4. Line Following and Tuning
5. Obstacle Avoidance
6. Competition

## Required Hardware

- ESP32
- L298N Motor Driver
- DC Motor
- Servo Motor
- 5 IR Sensors
- HC-SR04 Ultrasonic Sensor
- Battery Pack

Proceed to Module 1.
