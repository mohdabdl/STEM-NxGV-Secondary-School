# Module 2: Motor Control

## Learning Outcomes

Students should be able to:

- Explain PWM
- Control motor speed
- Move vehicle forward and backward

---

## Theory

A DC motor converts electrical energy into rotational motion.

The speed of the motor can be controlled using Pulse Width Modulation (PWM).

PWM ranges from:

0 = Stop
255 = Full Speed
